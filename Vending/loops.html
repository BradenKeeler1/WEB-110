Using loops with snacks

Question

for (let i = 1; i <= 5; i++) { console.log("Snack #" + i + " given to customer"); }

Use the code above to answer these questions:

1. How many times does the loop execute?

Answer: The loop executes 5 times because it starts at 1 and runs until i = 5.
